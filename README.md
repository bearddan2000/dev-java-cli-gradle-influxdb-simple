# dev-java-cli-gradle-influxdb-simple

## Description
Creates a small database table
called `dog` and populates.

## Tech stack
- java
- gradle

## Docker stack
- gradle:jdk11
- tutum/influxdb:latest

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
